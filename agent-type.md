# Tipo do Agente - agente-model

## Classificação

Agente conversacional com tool use e memória — padrão ReAct (Pensar → Agir → Observar → Repetir)

## O que NÃO é

- **Não é multi-agent** — não tem agentes especializados colaborando
- **Não é plan-and-execute** — não planeja antes de agir, reage em tempo real
- **Não é RAG puro** — usa memória semântica, mas não busca documentos externos
- **Não é autônomo** — só responde quando o usuário envia mensagem
